Instructions to turn on webcam and detect images:

1 - Open the anaconda command prompt
2 - Navigate to the "Mask_RCNN" folder
3 - Run the following command
	3.1 - python video_demo.py 0 (To use the algorithm via webcam)
	3.2 - python video_demo.py "Video file name or Photo name" (To read a video or photo)
